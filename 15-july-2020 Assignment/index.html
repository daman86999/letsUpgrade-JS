Array methods

pop() // The pop method removes the last element fro an arrays

var fruits = ["Banana", "Orange", "Apple", "Mango"];
fruits.pop();

push() //The push method adds an element to an array in the end

var fruits = ["Banana", "Orange", "Apple", "Mango"];
fruits.push("Kiwi");

shift() // method method removes first element of array and shifts other elements to left

var fruits = ["Banana", "Orange", "Apple", "Mango"];
fruits.shift();

unshift() //method adds a new element to array but this thime it adds it in front of the Array
The method takes 2 parameters: the start position, and the end position (end not included).

var fruits = ["Banana", "Orange", "Apple", "Mango"];
fruits.unshift("Lemon");
